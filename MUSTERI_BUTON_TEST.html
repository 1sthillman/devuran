🔥 Müşteri Bildirim Butonları - Test Rehberi

Garson, Köz, Hesap butonlarının çalışıp çalışmadığını test et

1️⃣ Müşteri Menüsünü Aç

http://localhost:3000/restaurant/nk5O1R45VhqxiB0FZTjr/table/U_0RDjoXmh

Veya alternatif masa:

http://localhost:3000/restaurant/nk5O1R45VhqxiB0FZTjr/table/g5J5476WuS

2️⃣ Browser Console'u Aç

Klavyede F12 tuşuna bas

Veya sağ tıklayıp "Inspect" → "Console" sekmesi

3️⃣ Console'da Bu Logları Gör

Sayfa yüklendiğinde:

🔔 ========== NotificationButtons BAŞLATILDI ==========
📊 Props: { restaurantId, tableId, tableName }
✅ Component mount edildi, butonlar görünmeli!

✅ Başarılı: Butonlar render olmuş

❌ Log yok: Component mount olmamış - CustomerMenu'de problem var

4️⃣ FAB Butonuna Tıkla

Sağ alt köşede turuncu yuvarlak buton

Yukarı ok ikonu ile

Tıklayınca 3 buton açılmalı:

  • 📞 Garson
  • 🔥 Köz
  • 💰 Hesap

5️⃣ "Köz" Butonuna Tıkla

Console'da şu logları göreceksin:

🔔 BİLDİRİM GÖNDERİLİYOR! {
  restaurantId: "nk5O1R45VhqxiB0FZTjr",
  type: "coal_request",
  tableId: "...",
  tableName: "..."
}

✅ BİLDİRİM OLUŞTURULDU: [notification-id]

6️⃣ Toast Mesajlarını Gör

Ekranda sırayla:

  • Loading: 🔥 Köz talebiniz gönderiliyor...
  • Success: 🔥 Köz talebiniz iletildi!

7️⃣ Garson Panelini Kontrol Et

http://localhost:3000/dashboard?tab=restaurant

"Yeni Köz Talepleri" bölümünde yeni bir kart görmelisin

Masa numarası ve talep doğru görünmeli

🐛 Sorun Giderme

Problem: Console'da log yok

Çözüm: NotificationButtons component render olmamış


Problem: "Permission Denied" hatası

Çözüm: Firestore rules deploy edilmemiş


Problem: "tableId is loading"

Çözüm: Biraz bekle, table verisi yükleniyor


Problem: Toast görmüyorum

Çözüm: Sonner toast provider eksik olabilir

✅ Başarılı Test Göstergeleri

  1. Console'da mount logu var
  2. FAB butonu görünür ve tıklanır
  3. 3 buton açılıyor (Garson, Köz, Hesap)
  4. Butona tıklayınca console'da "GÖNDERİLİYOR" logu
  5. Toast mesajları görünüyor
  6. Garson panelinde bildirim beliriyor

❌ Başarısız Test Göstergeleri

  1. Console'da hiç log yok → Component render olmamış
  2. FAB butonu görünmüyor → createPortal çalışmamış
  3. Butona tıklayınca hiçbir şey olmuyor → onClick handler bağlanmamış
  4. Permission denied → Firestore rules sorunu
