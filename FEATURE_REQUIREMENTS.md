## 🚀 HIZLI BAŞLANGIÇ

Eğer hemen başlamak istiyorsanız, önce şunları yapın:

### 1. Ödeme Entegrasyonu
```bash
# iyzico kurulumu
npm install iyzipay
```

### 2. Bildirim Sistemi
```bash
# Firebase Cloud Messaging
npm install firebase-admin
```

### 3. Email Servisi
```bash
# Nodemailer veya SendGrid
npm install nodemailer
```
