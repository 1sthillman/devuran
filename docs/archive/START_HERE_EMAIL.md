## ⚡ 5 Dakikada Başla

### 1️⃣ Resend Hesabı (2 dk)
```
https://resend.com
→ Sign Up
→ API Keys
→ Create API Key
→ Kopyala
```

### 2️⃣ API Key Ekle (1 dk)
```bash
firebase functions:secrets:set RESEND_API_KEY
# API key'i yapıştır
```

### 3️⃣ Deploy (2 dk)
```powershell
# Windows
.\deploy-email-functions.ps1

# Mac/Linux
./deploy-email-functions.sh
```

### 4️⃣ Test (30 sn)
```
Uygulamada randevu oluştur
→ Email'leri kontrol et
→ ✅ Çalışıyor!
```
